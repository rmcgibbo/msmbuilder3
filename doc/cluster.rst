Clustering
~~~~~~~~~~
.. currentmodule:: msmbuilder3.cluster

Clustering algorithms group a individual datapoints in a dataset into discrete clusters. In MSMBuilder3, this amounts to a ``Transform`` of multivariate timseries into a timeseries in a discrete space. Currently, only one clustering algorithim is here. Others will be implemented following the same API.
